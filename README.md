# bookFinderAPI
REST API for bookFinder application.
## Usage

### Clone
Clone the Project

```sh
$ git clone https://github.com/tanishha/bookFinderAPI.git
$ cd bookFinderAPI
```

### Installation

Install the dependencies

```sh
$ npm install
```

### Run

This will run your dev server at http://localhost:7070

```sh
$ npm start
```
